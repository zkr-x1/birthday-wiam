This is a special interactive birthday website created for Wiam.

## How to Customize

1.  **Add Photos**:
    *   Go to the `images` folder.
    *   Add 4 photos of Wiam (or more).
    *   Name them `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, and `photo4.jpg`.
    *   If you have different file names, update the `index.html` file where you see `<img src="images/photo1.jpg" ...>`.

2.  **Music (Optional)**:
    *   If you want background music, find a music file (MP3).
    *   Name it `music.mp3` and place it in the `birthday-wiam` folder.
    *   Uncomment the `<audio>` tag in `index.html`.

## How to Deploy

1.  Upload this entire folder to a GitHub repository.
2.  Go to the repository **Settings**.
3.  Scroll down to **Pages**.
4.  Select the `main` branch and click **Save**.
5.  GitHub will give you a link to share with Wiam!
